
# Drag and Drop Test App

Цей проект демонструє "сиру" реалізацію Drag & Drop у SwiftUI без використання сторонніх бібліотек.

## Мета
Виявити складнощі реалізації та виділити код для майбутнього фреймворку.

## Плановані фічі
- [x] Реалізувати DropDelegate
- [x] Зробити анімацію переміщення
- [x] Реалізувати перетягування між списками
